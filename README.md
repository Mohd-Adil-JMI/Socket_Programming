# Socket Programming

Multi client chat application with profanity filter built with Socekt.io(Node js)

## Installation

Use the node package manager [npm](https://nodejs.org/en/) to install.

```bash
npm install
```
Start Server
```bash
node server.js
```
Start Client
```bash
node client.js
```
